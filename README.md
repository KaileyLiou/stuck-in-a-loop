# Stuck in a loop

A small platformer made in GameMaker:
- The floor rises after each loop
- Spikes move with the platforms and will reset the player if touched
- Player can only win after completing 5 loops
- Includes basic screen shake after each platform rise
- Auto-resets if the player falls

## How to Play

Use the arrow keys to move up, left, down, and right. Use the space key to jump.
